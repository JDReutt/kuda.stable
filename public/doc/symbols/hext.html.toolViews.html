Namespace hext.html.toolViews

A set of functions for displaying HtmlViews. In order for the the ToolViews to be displayed, you need to have a div with the container id in your viewer page. The default container id that ToolViews searches for is 'hext-toolviews', but you may change the id if you wish.

  • Defined in: <js/hext/html/toolViews.js>
